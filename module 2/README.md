# Module 2 – Word Analysis

## Objective
The objective of this module is to analyze linguistic properties of text using NLP techniques such as regex processing, stemming, lemmatization, POS tagging, and CFG parsing.

---

## Description
This module focuses on word-level analysis and syntactic processing of product review text data.

---

## Tasks Performed

### 1. Regular Expression Analysis
Regex patterns were used to identify:
- Ratings
- Prices
- Emphasized words

Example:
- 4.5/5
- ₹999
- Amazing!!!

### 2. Stemming
Porter Stemmer was used to reduce words into root forms.

Example:
running → run

### 3. Lemmatization
WordNet Lemmatizer was used for meaningful root word extraction.

Example:
studies → study

### 4. POS Tagging
Parts of Speech tagging was performed to identify:
- Nouns
- Verbs
- Adjectives
- Adverbs

### 5. POS Pattern Observation
Important sentiment patterns identified:
- Adjectives indicate sentiment
- Adverbs intensify opinions
- Verbs indicate experiences

### 6. CFG Rules
Context Free Grammar rules were defined for sentence structure representation.

### 7. Parse Tree Representation
A basic parse tree was constructed to represent grammatical structure.

The module successfully performed:
