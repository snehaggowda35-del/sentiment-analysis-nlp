# NLP Sentiment Analysis Project

## Overview
This project is a Natural Language Processing (NLP) based Sentiment Analysis system developed using Python and Machine Learning.
The model analyzes product reviews and predicts whether the sentiment is:
- Positive
- Negative
- Neutral
The project was implemented and tested using Jupyter Notebook.

---
## Features

- Text preprocessing
- Tokenization
- Stopword removal
- TF-IDF vectorization
- Sentiment classification
- Neutral sentiment handling
- Product review prediction
---

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- NLTK
- Scikit-learn
- Matplotlib
---

## Workflow

1. Dataset Collection
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction using TF-IDF
5. Model Training
6. Sentiment Prediction
7. Model Evaluation
---
## Dataset
The dataset contains product reviews with labeled sentiments.
Example sentiments:
- Positive
- Negative
- Neutral
---
## Sample Prediction
### Input
```text
"The shoes I bought yesterday were fine, not like I hated it."
```
### Output
```text
Neutral
```
